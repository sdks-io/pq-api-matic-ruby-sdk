
# Hateoas Relationship

Indicates the HATEOS relationship between the target and current resources.

## Structure

`HateoasRelationship`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `rel` | `String` | Required | Indicates the relationship between the target and current resources.<br><br>**Default**: `'self'` |

## Example (as JSON)

```json
{
  "rel": "self"
}
```
